# English Project

An astronomy themed website made to present Indigenous Beliefs about Astronomy to an English Class.


## Features

- Interactive 3D Models to explore astronomy-themed environment.

- Educational Content to learn more about Indingeous beliefs about the cosmos.

- Animations to allow for smooth scrolling and a better user experience.

- 360 degrees free look by clicking and dragging to explore the environment.

## Tech Stack

- Next.js for server-side rendering.
- Tailwind CSSS for UI development.
- React to build user interfaces.
- SketchFab to find 3D models to integrate.
- Vercel to deploy and publish website online.



## How to Run

1. Install Dependencies:

npm install

2. Run the server:

npm run dev

3. Open http://localhost:3000/ to view it in your browser.
